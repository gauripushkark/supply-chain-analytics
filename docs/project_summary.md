# Project Summary

## Project
Supply Chain Analytics

## Purpose
This project simulates a supply chain operations environment and demonstrates how SQL-based analytics can be used to monitor shipment processing, backlog, SLA compliance, and facility performance.

## Business Value
The analysis helps operations teams identify delays, monitor pending work, compare facility performance, and improve decision-making around resource allocation and processing efficiency.

## Datasets Used
- shipments.csv
- inventory_items.csv

## Key KPIs
- SLA Compliance %
- Backlog Volume
- Processing Split
- Throughput
- Facility Performance
- Item Backlog by Category

## Tools
- SQL
- Python
- Power BI
- GitHub
